Mini FP set-up guide

How to proceed when a new customer should be set-up in FlashPost?

  • Always ask partners to provide you with all information in the new excel document - FlashPost template
  • Send a set-up request email to partners. This request should include:
    • Company information (address, contact person email, etc.)
    • Contract information (start date, duration, end date, product, credits, price)
    • Monthly PO info
    • Logo/template
    • Additional info if necessary
  • For the following 4 steps, see FP guide for Admins for further reference:
    • Create a new customer in FP
    • Create channels for this customer (! Remember: only enable the channel if you receive the codes
    • Import header for liquid design in case of customized jobs for DE and AT
    • Create a new wallet for this customer (! Remember: use the connection with SF to create your wallet whenever possible)
  • Once you have all the codes from partenrs in FP:
    • In case of AF - send the customer ID to the selling partner / job distributor
    • In case of RA - inform the partner, so they could create the users for this customer
