Resellers

What should be the procedure for SD Team when validating a PO of a reseller

  • Customer: always check that the end customer is correctly linked to the reseller via Parent Account field
  • Contract: always check the following
    • The Parent Account name is included in Comments field of the contract
    • The start and end date is the same as the contract of the reseller itself (you can find the contract info on the Parent Account)
    • The number of jobs is the actual number of jobs posted in FP and not number of credits in FP wallet
    • The price is correct when compared with the reseller card (you can find the reseller card on the Parent Account)
  • PO: always check if the Parent Account name is included in Justification field of the IN PO (important for reporting)

How to keep a track on credit consumption for resellers?

Reporting on resellers:

  • Monthly check on consumption via prepared reports
  • As there are no actual credits in SF, make sure that there are enough credits each month in FP
