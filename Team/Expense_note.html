Expense Note Procedure

When you need to expense something to the company (costs during company events, well-being allowance, etc.), you should follow this procedure:

  • Use this expense note Excel template (if you don’t have it saved yet)
    • In case of a first use, make sure to fill in Name, IBAN, and BIC; then save for next use
  • Fill in the current month in which you are expensing some costs (top right)
  • Start filling in the individual expenses you have:
    • Doc. number: Always start with number 1 and continue until the number of expenses you have
    • Date: Input the date when the expense occurred
    • Expenses: Indicate what the expense is related to (in most cases, it will be something like ‘TNW Workshop – Airport’ or ‘Well-being allowance’)
    • Comment: Input any details related to the expense, i.e., parking, breakfast, new running shoes, etc.
    • Personal car use: Should you use your car for a company event, you can expense the cost as well; simply input the number of km and the related cost
    • Foreign currency: In case your expense was done in a foreign currency, input the cost in the foreign currency (i.e., 25 USD) and check on OANDA what the exchange rate was on the given day
    • Amount EUR (column I):
      • If your expense was in EUR, simply input the amount in here
      • f your expense was in a foreign currency, input the EUR amount you got from OANDA
    • Amount (column L): Do not touch as this column is automated and it calculates the total amount that should be paid for the expense note (if you scroll down, you will see the total amount there)
  • There is no need to sign the expense note anymore
  • As for the documents proving the costs claimed in the expense note, you should:
    • Save each expense in a PDF document separately
    • You should name each document the same as you call it in column C, including the Doc. number (i.e., if the first entry in the expense note is 1. / 21.5.2025 / Management Forum – Airport, then the name of the document that you submit for this expense should be ‘1. Management Forum – Airport’
    • If you lose the receipt itself, you can use a screenshot of the actual payment from your online banking
  • Once the expense note and the supporting documents are ready, create a folder 'Exepnse Note - Your name', zip the file and send them to ORN email
